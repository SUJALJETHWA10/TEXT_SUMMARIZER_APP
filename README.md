# 📝 Text Summarizer App Using T5 Transformer

A powerful **AI-powered Text Summarization Web Application** built using **FastAPI**, **Hugging Face Transformers**, and **T5 (Text-to-Text Transfer Transformer)**. This application converts long text into concise and meaningful summaries while preserving important information.

The project leverages the **T5 Transformer Model** for abstractive text summarization, enabling users to quickly understand lengthy articles, paragraphs, or documents.

---

## 🚀 Features

✅ AI-powered text summarization
✅ FastAPI-based web application
✅ Hugging Face Transformer integration
✅ T5 Pre-trained Model for abstractive summarization
✅ Clean and responsive UI using Jinja2 Templates
✅ Fast and efficient text processing

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Text-Summarizer-App.git
cd Text-Summarizer-App
```

---

### 2️⃣ Create Virtual Environment

For Python 3.10:

```bash
py -3.10 -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

Install required packages:

```bash
pip install -r requirements.txt
```

Or manually install:

```bash
pip install fastapi uvicorn transformers torch jinja2 python-multipart
```

---

## ▶️ Run the Application

Start the FastAPI server using Uvicorn:

```bash
uvicorn app:app --reload
```

The application will run locally at:

```bash
http://127.0.0.1:8000
```

## 🧠 How It Works

1. Enter or paste long text into the input box.
2. Click the **Summarize** button.
3. The T5 Transformer model processes the text.
4. Important information is extracted.
5. A concise summary is generated and displayed instantly.

---

## 🤖 Model Used

This project uses the **T5 (Text-to-Text Transfer Transformer)** model from Hugging Face.

Example model loading:

```python
from transformers import T5ForConditionalGeneration, T5Tokenizer

model = T5ForConditionalGeneration.from_pretrained("t5-small")
tokenizer = T5Tokenizer.from_pretrained("t5-small")
```

### Why T5?

* High-quality abstractive summarization
* Better sentence generation
* Context-aware summaries
* State-of-the-art NLP architecture
